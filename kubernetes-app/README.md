# Kubernetes Application - Fast Food

Aplicação principal do sistema Fast Food para pedidos e autoatendimento.

## Requisitos

- Node.js 22
- Docker
- Kubernetes (para deploy local)

## Instalação

```bash
npm install
```

## Testes

```bash
npm test
```

## Execução local

```bash
npm start
```

## Build Docker

```bash
docker build -t fast-food-app:latest .
```
